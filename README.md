# MiniAHU Selector

Веб-приложение для подбора приточных и приточно-вытяжных установок серий MiniAHU. Работает полностью на клиенте (без бэкенда).

## Функционал

- **Подбор моделей** — ручной (серия + типоразмер) или автоматический по расходу и давлению
- **24 серии установок** с базой данных в формате JSON, извлечённой из Excel
- **Аэродинамический расчёт** — характеристика вентилятора (полином 3-й степени), характеристика сети, рабочая точка
- **Расчёт рекуператора** — пластинчатый (КПД по теплу) и роторный (КПД по теплу и влаге), поправка на неравные потоки
- **Расчёт нагревателя** — электрический (тепловой баланс), водяной (метод ε-NTU)
- **График** — кривая вентилятора, кривая сети, рабочая точка, проектная точка (Chart.js)
- **Предупреждения** — обмерзание рекуператора, недостаточная мощность, выход за диапазон и т.д.
- **Опции** — регулятор скорости, клапан, привод, гибкие вставки, шумоглушитель, датчик давления
- **Экспорт CSV** и **Печать**

## Стек технологий

- React 19 + TypeScript
- Vite 8
- Zustand (состояние)
- Chart.js + react-chartjs-2
- Tailwind CSS v4

## Локальный запуск

```bash
# Клонировать репозиторий
git clone https://github.com/kryukovdaniiljob-cell/miniahu-selector.git
cd miniahu-selector

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
# → http://localhost:5173

# Сборка для продакшена
npm run build
# → dist/
```

## Деплой

### Vercel (рекомендуется)

1. Зайдите на [vercel.com](https://vercel.com) и создайте аккаунт
2. Нажмите **Add New → Project** и импортируйте репозиторий с GitHub
3. Оставьте настройки по умолчанию (Vercel автоматически определит Vite)
4. Нажмите **Deploy**

### GitHub Pages

В `vite.config.ts` установите `base` в название репозитория:

```typescript
base: '/miniahu-selector/'
```

Затем добавьте в `package.json` скрипт и задеплойте:

```bash
npm install -D gh-pages
npx gh-pages -d dist
```

## Структура проекта

```
src/
  data/
    models/          # 24 JSON-файла с данными установок
    modelLoader.ts   # Загрузка и классификация серий
  calculations/
    aero.ts          # Аэродинамика (рабочая точка)
    recuperator.ts   # Рекуператор (пластинчатый / роторный)
    heater.ts        # Нагреватель (электрический / водяной ε-NTU)
    selector.ts      # Главный оркестратор расчёта
    validation.ts    # Валидация входных данных
    utils.ts         # Вспомогательные функции
  components/
    FormPanel.tsx    # Форма ввода параметров
    ResultTable.tsx  # Таблица результатов
    Graph.tsx        # График аэродинамики
    Warnings.tsx     # Блок предупреждений
  store/
    useStore.ts      # Zustand-хранилище
  types/
    index.ts         # TypeScript-интерфейсы
  App.tsx
```

## Тестовый сценарий

Настройки по умолчанию соответствуют тестовому расчёту:

| Параметр | Значение |
|---|---|
| Серия | Unimax_P_CE |
| Типоразмер | 4 (Unimax P 1500 CE) |
| Расход приток | 500 м³/ч |
| Давление | 150 Па |
| Наружная температура | −30 °C |
| Влажность наружная | 80% |
| Температура вытяжки | 18 °C |
| Требуемая температура приточного воздуха | 21 °C |

Ожидаемые результаты:

| Параметр | Значение |
|---|---|
| Давление вент. при расч. Q | ≈ 432 Па |
| Температура после рекуп. | ≈ −4 °C |
| КПД рекуператора | ≈ 54% |
| Требуемая мощность нагревателя | ≈ 3.6 кВт |
